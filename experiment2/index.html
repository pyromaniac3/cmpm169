Experiment 2 - Vector Art, Animation, and Interactivity

Imitate

I started with a simple Perlin Noise code that made a pretty wave of lines horizontally across the screen. In this project I knew I wanted to do make something pretty using this alogorithm.

Integrate

My overall goal was to create this counter rotating noise generation, in which I would have 2 shapes of those curving lines spinning around each other.
So to start that process I need to:

  • Create two perpendicular sets of noise lines.
  • Make the screen feel less cluttered. I noticed after a while, it started to become a little overwhelming. I can do this either by allowing the player to reset or create some sort of fade.
  • Rotate the lines in some circular fashion.

    • The hardest task was the rotation. To create the two perpendicular lines, all I had to do was switch x and y values. I created a fade using a mostly transparent background to draw between each frame. But to rotate the shapes, I had to push the current state of the shape, transform it acorss the screen and rotate it, then push to get the previous state back, before moving on to the next vertex being made.
    With the help of ChatGPT I was able to get it to work in the end.

    Innovate

    How did you use the code as a jumping off point to create something truly unique and novel? Document this step thoroughly. Most of my innovation came from just adding interactivity to the project. I made it so you could change the color for every new shape made, and create more overlapping shapes with the press of the button.
